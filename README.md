# Book Management App
A React application built to master **Lifting State** and **Unidirectional Data Flow**.

### Features:
- Filterable book list via Search bar.
- "Hover to Add" interactive buttons.
- Global Bookshelf state managed in the Parent component.
- Persistence using LocalStorage.

### Lessons Learned:
- How to pass functions as props to update parent state from child components.
- Managing arrays in React state using immutability.
